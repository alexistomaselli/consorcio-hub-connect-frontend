Guía de Instalación

Requisitos Previos

1

Instalar Dependencias

  • Node.js ≥ 18.x
  • npm ≥ 9.x
  • PostgreSQL ≥ 14
2

Clonar e Instalar

$ git clone https://github.com/your-repo/consorcio-hub-connect.git $ cd consorcio-hub-connect $ npm install
3

Configurar Variables de Entorno

Crea un archivo .env en la raíz del proyecto:

Variable Descripción
DATABASE_URL URL de conexión a PostgreSQL
JWT_SECRET Clave secreta para JWT
PORT Puerto para el servidor (default: 8080)

Scripts Disponibles

Comandos NPM

# Desarrollo
$ npm run dev      # Inicia servidor de desarrollo
$ npm run build    # Construye para producción
$ npm run preview  # Vista previa de producción

# Base de datos
$ npm run db:migrate  # Ejecuta migraciones
$ npm run db:seed     # Carga datos iniciales

# Testing
$ npm run test     # Ejecuta tests
$ npm run test:e2e # Tests end-to-end

Herramientas de Desarrollo

Prisma Studio

Prisma Studio es una interfaz gráfica para gestionar la base de datos. Permite:

  • Ver y editar datos en todas las tablas
  • Crear nuevos registros
  • Filtrar y ordenar datos
  • Visualizar relaciones entre tablas
# Iniciar Prisma Studio $ cd backend && npx prisma studio # Acceder vía navegador URL: http://localhost:5555

PgAdmin

PgAdmin es una plataforma de administración para PostgreSQL que ofrece:

  • Interfaz visual para gestionar bases de datos
  • Herramientas de consulta SQL
  • Gestión de usuarios y permisos
  • Monitoreo de rendimiento
Acceso a PgAdmin:
URL: http://localhost:5050
Email: admin@admin.com
Password: admin
