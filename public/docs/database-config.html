Configuración de Base de Datos

Arquitectura de Docker

La arquitectura de nuestra base de datos en Docker consiste en dos contenedores principales:

  • PostgreSQL: Ejecutándose en el puerto interno 5432
  • pgAdmin: Interfaz web para administrar la base de datos

Mapeo de Puertos

PostgreSQL

Puerto Interno (Contenedor)

5432

Puerto Externo (Host)

5433

Conexión a la Base de Datos

Desde la Aplicación

DATABASE_URL="postgresql://admin:admin123@localhost:5433/consorcio_hub"

Usa el puerto externo (5433) porque se conecta desde fuera de Docker

Desde pgAdmin

Host: postgres
Port: 5432
Database: consorcio_hub
Username: admin
Password: admin123

Usa el puerto interno (5432) porque está dentro de la red Docker

Acceso a pgAdmin

Para acceder a la interfaz web de pgAdmin:

  1. Abre http://localhost:5051
  2. Usa las siguientes credenciales:
    • Email: admin@admin.com
    • Password: admin123
